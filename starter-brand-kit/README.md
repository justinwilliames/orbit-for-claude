# Starter Brand Kit

Copy this folder into your own brand asset directory and replace the placeholder paths with real files.

## Intended Structure

```text
your-brand-kit/
  brand-profile.json
  brand-guidelines.md
  logos/
    primary-logo.svg
    alternate-logo.svg
  examples/
    email-header-1.png
    email-header-2.png
    campaign-reference-1.png
```

## Minimum Requirements

- one official logo
- at least two example assets
- a `brand-profile.json`
- a `brand-guidelines.md`
- a simple brand color map

## Notes

- Keep the official logo as a real asset file. Orbit will composite it locally.
- Add example assets that show real brand tone, spacing, and texture.
- Use hex colors in `brand-profile.json` for the cleanest results.
- Keep `brand-guidelines.md` as the editable longform source for visual rules, tone of voice, and email-header guidance.

Orbit bundles these self-hosted fonts for internal Orbit artifacts only.

- `Sora`: primary UI font for Orbit preview shells and internal labeling
- `Oxanium`: display/accent font for Orbit headings, badges, and diagram labels
- `Geist Mono`: monospace font for Orbit metadata and technical labels

These fonts are intentionally not injected into customer-facing email HTML.

Upstream sources:
- `Sora`: Google Fonts
- `Oxanium`: Google Fonts
- `Geist Mono`: Vercel Geist
